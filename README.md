## 🌟 About

**PassGuard** is a simple, secure, and privacy-first web tool that helps you check if your passwords have been exposed in any data breaches. 

Unlike other password checkers, **PassGuard NEVER sends your actual password over the internet**. Using industry-standard k-Anonymity and SHA-1 hashing, your password remains on your device at all times.

### Why PassGuard?

- 🔒 **Zero Password Transmission** - Your password never leaves your browser
- 🛡️ **k-Anonymity** - Only 5 characters of the hash are sent to the API
- 🎯 **Instant Results** - Check against 500+ million leaked passwords
- 💰 **100% Free** - No signups, no ads, no tracking
- 🌐 **Works Offline** - Password strength and generation work without internet
- 📦 **Single File** - Just one HTML file, no installation needed

## ✨ Features

### 🔑 Single Password Check
Check one password at a time with detailed results and security tips.

### 📝 Multiple Passwords
Check multiple passwords at once by typing them line by line.

### 📂 File Upload
Upload a `.txt` file containing passwords and check them all instantly.

### 💪 Password Strength Meter
Real-time feedback on password strength with color-coded indicators:
- 💪 **Strong** (80-100%)
- 👍 **Good** (60-79%)
- ⚠️ **Fair** (40-59%)
- 🔴 **Weak** (0-39%)

### 🎲 Password Generator
Create strong, unique passwords with adjustable length (8-32 characters).

### 📊 Statistics Tracking
Track your password checks:
- Total passwords checked
- Number of leaked passwords found
- Number of safe passwords
- Leak percentage

### 👁️ Show/Hide Password
Toggle password visibility for convenience and security.

### 📋 Copy to Clipboard
One-click copying for generated passwords.

### 🎨 Modern UI
- Dark theme
- Smooth animations
- Fully responsive
- Works on all devices
